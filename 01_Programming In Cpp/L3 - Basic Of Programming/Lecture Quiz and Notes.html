Unary Operators

‚ûï‚ûñ Unary Operators

Unary operators operate on a single operand. In C++, these include the increment (++) and decrement (--) operators. These can be used in two forms:

üîµ Pre-increment/decrement

Alters the operand before its value is used in an expression.

Example: ++a or --a

üü° Post-increment/decrement

Alters the operand after its value is used in an expression.

Example: a++ or a--

üíª Unary Operators Example 
int a = 5;
int b = ++a; // Pre-increment: a is 6, b is 6
int c = b--; // Post-decrement: b is 6 (then becomes 5), c is 6
‚ö†Ô∏è Important!

Understanding these distinctions is vital as they affect the flow and performance of your code.

Operator Precedence and Associativity

üìä Precedence and Associativity
üéØ Operator Precedence

Determines the order in which operators are evaluated in an expression.

Example: In a + b * c, multiplication is evaluated first.

üîÑ Associativity

Dictates the order of evaluation for operators of the same precedence level:

  • Left-to-right: Most operators in C++ (e.g., +, -, *, /)
  • Right-to-left: Assignment operators (=, +=, -=, etc.)
‚ö†Ô∏è Best Practice

Understanding precedence and associativity is crucial, especially in complex expressions. Consider using parentheses to make the order of evaluation explicit, improving both readability and bug reduction.

Conditionals

üîÄ Conditional Statements

Conditionals allow your program to make decisions:

üìå if block

Basic structure to execute code if a condition is true.

üìå if-else block

Provides an alternative path if the condition is false.

üìå if-else if-else ladder

Evaluates multiple conditions in sequence.

üìå switch case

Suitable for selecting between multiple paths based on the value of a variable.

Remember:
  • Use break to prevent fall-through
  • The default case is optional but useful for handling unexpected values
üíª Switch Case Example 
switch (variable) {
    case 1:
        // Code block
        break;
    case 2:
        // Code block
        break;
    default:
        // Code block
}

Ternary Operator

‚ùì Ternary Operator

A compact form of if-else:

üíª Ternary Operator Syntax 
int result = (condition) ? value_if_true : value_if_false;

üí° Usage: This operator is useful for simple conditional assignments and improves code readability when used appropriately.

Loops and Special Cases

üîÑ Loops and Infinite Loop

The for loop is a powerful construct that iterates a block of code. A special case is for(;;), which generates an infinite loop.

‚ö†Ô∏è Controlling Infinite Loops:

Such loops can be controlled using break or return statements to prevent them from running endlessly.

üíª Infinite Loop Example 
for (;;) {
    // Infinite loop body
    if (stop_condition) {
        break;
    }
}

Understanding: Through these discussions, students understand not only the syntax but also the conceptual understanding of how these elements affect the program's flow and logic.

Explanation: A for(;;) loop in C++ is an infinite loop. This is because the loop has no initialization, condition, or increment clause, causing it to run indefinitely until externally interrupted or a break/exit condition within the loop body is met.
