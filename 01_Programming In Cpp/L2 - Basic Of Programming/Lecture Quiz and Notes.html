6Ô∏è‚É£ Types of Variables: Local vs Global Scope
üî∑ Local Scope:
  • Accessible only within the function/block they are declared
  • Created when function is called, destroyed when function ends
  • Example: Variables declared inside main()
üî∂ Global Scope:
  • Declared outside all functions
  • Accessible throughout the entire program
  • Exists for the lifetime of the program
