  • Sum all these values to get the decimal equivalent.

2Ô∏è‚É£ Decimal to Binary Conversion

Method
  • Divide the number by 2.
  • Write down the remainder.
  • Continue with the quotient until it is zero.
  • The binary number is the sequence of remainders read from bottom to top.
üíª Example: Convert Decimal 13 to Binary 
13 / 2 = 6 remainder 1
6 / 2 = 3 remainder 0
3 / 2 = 1 remainder 1
1 / 2 = 0 remainder 1

Binary representation: Read remainders from bottom to top = 1101

üí° Remember:

When converting from decimal to binary, always read the remainders from bottom to top. This is the most common mistake students make!
