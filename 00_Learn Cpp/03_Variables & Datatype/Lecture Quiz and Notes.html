What is a Variable?

A Variable in C++ is a storage unit in memory to hold data. It acts as a container for information that can be manipulated within a program.

Understanding variables involves knowing how to declare, define, and initialize them.

üìù Declaration
Announcing a variable without value
üîß Definition
Allocating memory and assigning value
üîÑ Re-assignment
Updating with new values

Variable Operations:

1. Declaration: Announcing a variable to the compiler without assigning a value:
Declaration Example 
int age;
2. Definition and Initialization: Allocating memory and assigning an initial value simultaneously:
Initialization Example 
int age = 25;
3. Re-assigning Values: Variables can be updated with new values:
Re-assignment Example 
age = 30;

Question 1
Which of the following correctly represents variable declaration in C++?
A. int a = 10;
B. int a; ‚úì

Explanation: In C++, a variable declaration involves specifying the type and the name of the variable. 'int a;' declares a variable 'a' of type int without assigning an initial value.
Question 2
Which datatype in C++ is used to represent single characters and what is its typical size?
A. char, 1 byte ‚úì
B. char, 2 bytes
C. string, 1 byte
D. string, 2 bytes
Explanation: The 'char' datatype in C++ is used to store single characters and typically occupies 1 byte of memory.
Question 3
What will be the output of the following C++ code?
#include <iostream>
using namespace std;
int main() {
    int a = 5, b = 10;
    a = b;
    b = 20;
    cout << a << " " << b;
    return 0;
}
A. 5 20
B. 10 20 ‚úì
C. 5 10
D. 20 10
Explanation: Initially, 'a' is 5 and 'b' is 10. After 'a = b;', 'a' becomes 10. Re-assigning 'b = 20;' doesn't affect 'a', so the output is '10 20'.
Question 4
What is the correct way to initialize a float variable in C++?
A. float f = '3.14';
B. float f = 3.14;
C. float f = 3.14f; ‚úì
D. float f = "3.14";
Explanation: The 'f' at the end of '3.14f' indicates that it is a float literal, which is required to explicitly initialize a float variable in C++. Without the 'f', it is considered a double.
