Do-While and Nested Loops in C++

Advanced Loop Structures for Complex Iterations

Overview: In this session, we explore the do-while loop and nested loops. These are fundamental constructs in C++ programming, particularly when dealing with iterative processes. Understanding these structures is crucial for writing efficient and readable code.

Do-While Loop

üîÅ Do-While Loop

The do-while loop executes a block of code at least once, and then continues executing as long as a specified condition is true. This is particularly useful when the loop needs to run at least once regardless of the condition.

üìã Syntax:
do {
    // Code block to be executed
} while (condition);
üîë Key Components:
  • The keyword do precedes the code block to be executed
  • A while statement follows the block, containing the condition to check
  • The loop will execute as long as the condition evaluates to true
  • Important: The condition is checked after the loop runs
üíª Do-While Loop Example 
int i = 0;
do {
    cout << i << " ";
    i++;
} while (i < 5);

Output: 0 1 2 3 4

Explanation: In this example, numbers 0 to 4 will be printed. Note that the condition is checked after the loop runs, ensuring the block executes at least once.

‚öñÔ∏è Do-While vs While Loop
While Loop:
  • Condition checked before execution
  • May execute zero or more times
  • Use when unsure if loop should run
Do-While Loop:
  • Condition checked after execution
  • Always executes at least once
  • Use when loop must run at least once

Nested Loops

üîÑüîÑ Nested Loops

Nested loops are loops within loops, allowing us to execute a sequence of operations several times over different sets of data.

üíª Nested Loop Example 
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 2; j++) {
        cout << "i: " << i << ", j: " << j << endl;
    }
}
üìä Iteration Breakdown
Explanation:
  • The outer loop (for i) runs 3 times
  • For each iteration of the outer loop, the inner loop (for j) runs 2 times
  • This results in a total of 6 iterations (3 √ó 2 = 6)
  • This demonstrates the multiplication of iterations typical of nested loops
Output: 
i: 0, j: 0
i: 0, j: 1
i: 1, j: 0
i: 1, j: 1
i: 2, j: 0
i: 2, j: 1
üí° Understanding Nested Loops:

Think of nested loops like a clock: the inner loop completes all its iterations (like seconds) before the outer loop advances one step (like minutes). For every outer loop iteration, the inner loop runs completely from start to finish.

Common Mistakes

Be aware of these common pitfalls when working with do-while and nested loops:

‚ùå 1. Infinite Loops

Ensure the loop's condition eventually evaluates to false.

// Wrong: Infinite loop
int i = 0;
do {
    cout << i;
    // Forgot to increment i!
} while (i < 5);

// Correct: Proper increment
int i = 0;
do {
    cout << i;
    i++;  // Don't forget this!
} while (i < 5);
‚ùå 2. Off-by-One Errors

Remember that loops like for and while use zero-based counting.

// Wrong: Runs 6 times instead of 5
do {
    cout << i;
    i++;
} while (i <= 5);

// Correct: Runs exactly 5 times
do {
    cout << i;
    i++;
} while (i < 5);
‚ùå 3. Misplaced Semi-Colons

Avoid placing a semicolon directly after the condition or loop header, as it ends the loop statement unexpectedly.

// Wrong: Semicolon ends the loop immediately
while (i < 5);
{
    cout << i;
    i++;
}

// Correct: No semicolon after while
while (i < 5)
{
    cout << i;
    i++;
}
‚ùå 4. Using Same Variable in Nested Loops

In nested loops, never use the same loop variable for both loops as it causes unexpected behavior.

// Wrong: Same variable 'i' used in both loops
for (int i = 0; i < 3; i++) {
    for (int i = 0; i < 2; i++) {  // Error!
        cout << i;
    }
}

// Correct: Different variables
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 2; j++) {
        cout << "i: " << i << ", j: " << j;
    }
}
